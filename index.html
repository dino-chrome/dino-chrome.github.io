Dinosaur Game

Description

The Dinosaur Game is a classic endless runner where you guide a T-Rex through a desert, dodging obstacles like cacti and pterodactyls. Originally a hidden Easter egg in Google Chrome's offline mode, it has become a beloved game for its simplicity and addictive gameplay.

Instructions

Jump and duck to avoid obstacles. Survive as long as possible!

PC Controls

  • Spacebar = Jump
  • Down Arrow = Duck
